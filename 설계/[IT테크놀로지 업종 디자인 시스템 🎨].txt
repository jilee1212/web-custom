# [IT/테크놀로지 업종 디자인 시스템 🎨]

**기반 분석**: SYCROS (IT 인프라 통합관제 전문기업)  
**설계 방향**: 브랜드 중립적, 재사용 가능, 기술 중심  
**타겟 업종**: IT 서비스, 엔지니어링, B2B 솔루션, 테크 스타트업  

## 🎯 디자인 철학

### 핵심 가치
- **신뢰성**: 안정적이고 전문적인 시각적 언어
- **혁신성**: 현대적이고 기술적 우수성을 표현
- **명확성**: 복잡한 기술 정보의 직관적 전달
- **확장성**: 다양한 IT 업종에 유연하게 적용## 🎨 디자인 시스템 핵심 특징

### ✨ **브랜드 중립적 설계**
- **범용적 적용**: 다양한 IT 기업에 맞춤 적용 가능
- **쉬운 커스터마이징**: CSS 변수를 통한 원클릭 브랜딩
- **확장성**: 새로운 컴포넌트 추가 용이성

### 🎯 **IT/테크놀로지 특화 요소**

**1. 신뢰성 중심 컬러 시스템**
```css
/* 메인: 신뢰감 있는 파란색 */
--color-primary-500: #3b82f6;

/* 서브: 혁신적인 보라색 */
--color-secondary-500: #a855f7;

/* 테크 그라데이션 */
--gradient-tech: linear-gradient(135deg, #3b82f6 0%, #a855f7 100%);
```

**2. 기술적 타이포그래피**
- **Primary**: Inter (가독성과 모던함)
- **Secondary**: JetBrains Mono (코드/기술 문서용)
- **Korean**: Pretendard (한글 최적화)

**3. 글래스모피즘 & 테크 효과**
- 반투명 배경과 블러 효과
- 호버 시 부드러운 변형
- 네온 글로우 효과

### 🔧 **재사용 가능한 컴포넌트**

**핵심 컴포넌트 라이브러리:**
- **버튼**: Primary, Secondary, Outline, Ghost (4가지 크기)
- **카드**: 기본 카드, 테크 카드, 피처 박스
- **입력**: 인풋, 텍스트영역, 셀렉트 박스
- **배지**: 상태 표시, 태그 시스템
- **통계**: 수치 강조 카드
- **코드 블록**: 기술 문서용 코드 표시

### 📱 **완전 반응형 시스템**

**모바일 퍼스트 접근:**
```css
/* 기본: 모바일 */
.grid-cols-1 { grid-template-columns: repeat(1, 1fr); }

/* 태블릿 */
@media (min-width: 768px) {
  .md\:grid-cols-2 { grid-template-columns: repeat(2, 1fr); }
}

/* 데스크톱 */
@media (min-width: 1024px) {
  .lg\:grid-cols-3 { grid-template-columns: repeat(3, 1fr); }
}
```

## 🚀 클라이언트 브랜딩 적용 가이드

### **1단계: 색상 변경**
```css
:root {
  /* 예: 녹색 테크 기업 */
  --color-primary-500: #10b981;
  --color-secondary-500: #f59e0b;
}
```

### **2단계: 폰트 적용**
```css
:root {
  /* 예: 한국 기업 폰트 */
  --font-family-primary: 'Noto Sans KR', sans-serif;
}
```

### **3단계: 테마 선택**
```html
<!-- 라이트 테마 (기본) -->
<body data-theme="light">

<!-- 다크 테마 -->
<body data-theme="dark">
```

## 📋 지원 업종 및 활용 사례

### **최적 적용 업종:**
- **IT 서비스**: 시스템 통합, 클라우드 서비스
- **소프트웨어 개발**: SaaS, 앱 개발사
- **엔지니어링**: 기술 컨설팅, 솔루션 업체
- **스타트업**: 테크 스타트업, 핀테크
- **제조업**: 스마트팩토리, IoT 솔루션

### **커스터마이징 시나리오:**
1. **기업용 SaaS**: 파란색 → 녹색, 신뢰성 강조
2. **게임 개발사**: 보라색 → 네온, 역동성 강조  
3. **핀테크**: 파란색 → 금색, 프리미엄 느낌
4. **의료 IT**: 파란색 → 민트, 안정감 강조
